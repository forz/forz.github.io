LeetCode 205. Isomorphic Strings

Given two strings s and t, determine if they are isomorphic.

Two strings are isomorphic if the characters in s can be replaced to get t.

All occurrences of a character must be replaced with another character while preserving the order of characters. No two characters may map to the same character but a character may map to itself.

For example,

Given “egg”, “add”, return true.

Given “foo”, “bar”, return false.

Given “paper”, “title”, return true.

Note: You may assume both s and t have the same length.

解析：

本题要求判断同构字符串，可以用两个hash表来做。hash表中存放上一个相同字符的位置。每次两个字符串比较其hash表中对应的值即可判断是否同构。这里有一个技巧，哈希表存放的是当前位置+1，这样就可以用数值0表示没出现过的字符了。

class Solution {
public:
bool isIsomorphic(string s, string t) {
    int m1[256] = {0}, m2[256] = {0}, n = s.size();//制作哈希表
    for (int i = 0; i < n; ++i) {
        if (m1[s[i]] != m2[t[i]]) return false;//字符串满足双射关系，位置保持完全一致
        m1[s[i]] = i + 1;//哈希表存放当前位置+1
        m2[t[i]] = i + 1;
    }
    return true;
}
};

LeetCode 290. Word Pattern

Given a pattern and a string str, find if str follows the same pattern.

Here follow means a full match, such that there is a bijection between a letter in pattern and a non-empty word in str.

Examples:

pattern = “abba”, str = “dog cat cat dog” should return true.

pattern = “abba”, str = “dog cat cat fish” should return false.

pattern = “aaaa”, str = “dog cat cat dog” should return false.

pattern = “abba”, str = “dog dog dog dog” should return false.

Notes:

You may assume pattern contains only lowercase letters, and str contains lowercase letters separated by a single space.

解析：

字母和字符串要双射，保存两者的位置是最好的.

class Solution {
public:
    bool wordPattern(string pattern, string str) {
        map<char, int> p2i;//字母对应的位置
        map<string, int> w2i;//字符串对应的位置
        istringstream in(str);
        int i = 0, n = pattern.size();
        for (string word; in >> word; ++i) {
            if (i == n || p2i[pattern[i]] != w2i[word])
                //每次i都会加1，如果str比pattern串长，此时i==n，否则i<n
                return false;
            p2i[pattern[i]] = w2i[word] = i+1;//i+多少无所谓
        }
        return i == n;//保证str不必pattern串短
    }
};
