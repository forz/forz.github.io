数字字符串转换为字母组合的种数

给定一个字符串str, str全部由数字字符组成，如果str中某一个或某相邻两个字符组 成的子串值在1〜26之间，则这个子串可以转换为一个字母。规定"1"转换为"A”，“2"转换为"B”,3"转换为"C",“26”转换为"Z”。写一个函数，求str有多少种不同的转换结果，并返回种数。

解析:

以上过程中，p(i)最多可能会有两个递归分支P(i+1)和p(i+2),一共有N层递归，所以时间复杂度为0(2^N),额外空间复杂度就是递归使用的函数栈的大小为O(N)。但是研究一 下递归函数就会发现，P(i)最多依赖P(i+1)和p(i+2)的值，这是可以从后往前进行顺序计算的，也就是先计算p(N)和p(N-1),然后根据这两个值计算p(N-2),再根据p(V-1)和P(N-2)计算p(N-3)，最后根据p(1)和p(2)计算出p(0)即可，类似斐波那契数列的求解过程，只不过斐波那契数列是从前往后计算的，这里是从后往前计算而已。

因为是顺序计算，所以num2方法的时间复杂度为O(N),同时只用了cur、next和tmp 进行滚动更新，所以额外空间复杂度为0(1)。但是本题并不能像斐波那契数列问题那样用 矩阵乘法的优化方法将时间复杂度优化到O(log^),这是因为斐波那契数列是严格的 f(n)=f(n+1)+f(n+2)，但是本题并不严格，Str[i]的具体情况决定了f(i)是等于0还是等于f(n+1)+f(n+2)，还是等于f(n+1)。

	public static int num2(String str) {
		if (str == null || str.equals("")) {
			return 0;
		}
		char[] chs = str.toCharArray();
		int cur = chs[chs.length - 1] == '0' ? 0 : 1;
		int next = 1;
		int tmp = 0;
		for (int i = chs.length - 2; i >= 0; i--) {
			if (chs[i] == '0') {
				next = cur;
				cur = 0;
			} else {
				tmp = cur;
				if ((chs[i] - '0') * 10 + chs[i + 1] - '0' < 27) {
					cur += next;
				}
				next = tmp;
			}
		}
		return cur;
	}

LeetCode 343. Integer Break

Given a positive integer n, break it into the sum of at least two positive integers and maximize the product of those integers. Return the maximum product you can get.

For example, given n = 2, return 1 (2 = 1 + 1); given n = 10, return 36 (10 = 3 + 3 + 4).

Note: You may assume that n is not less than 2 and not larger than 58.

Credits: Special thanks to @jianchao.li.fighter for adding this problem and creating all test cases.

解析：

由均值不等式(n个数的算术平均数大于等于它们的几何平均数)得：当把输入的n拆分成几个相等的数时它们的积最大。

为了方便使用导数，我们先假设我们可以把n拆分成实数。那么设每一个数为x,则一共有n/x个数。设它们的积为f(x),则f(x)=x(n/x)，那么怎么求f(x)最大值呢？求导数！

f′(x)=(n/x2) x(n/x) (1-lnx)

当x=e时取极大值。而我们题目里规定x为整数，那么我们只需要取的x越靠近e越好。那么2<e<3，而且e=2.71828…，所以取3是最好的，如果取不到3就取2。

class Solution {
public:
    int integerBreak(int n) {
        if(n < 4) return n-1;
        int res = 1;
        while(n > 2){//看n包含多少个3,把他们相乘,直到n<=2
            res *= 3;
            n -= 3;
        }
        if(n == 0) return res;//n可以整除3，res就是各个3相乘
        if(n == 1) return (res / 3 ) * 4;//除3余1，把其中的一个3加1变为4再相乘
        if(n == 2) return res * 2;//除3余2,则可直接把2与res相乘
    }
};

补充：常规DP方法

令dp[n]为n对应的最大积。

那么递推方程就是：dp[n]=max(i*dp[n-i],i*(n-i))(其中i从1到n-1)。

class Solution {
public:
    int integerBreak(int n) {
        vector<int>dp(n+5,-1);
        dp[1]=1;
        dp[2]=1;
        for(int i=3;i<=n;i++)
            for(int j=1;j<i;j++)
                dp[i]=max(j*dp[i-j],max(dp[i],j*(i-j)));
        return dp[n];
    }
};

LeetCode 139. Word Break

Given a non-empty string s and a dictionary wordDict containing a list of non-empty words, determine if s can be segmented into a space-separated sequence of one or more dictionary words. You may assume the dictionary does not contain duplicate words.

For example, given。
s = “leetcode”,
dict = [“leet”, “code”].

Return true because “leetcode” can be segmented as “leet code”.

UPDATE (2017/1/4):
The wordDict parameter had been changed to a list of strings (instead of a set of strings). Please reload the code definition to get the latest changes.

解析：

判断2个单词能否组成1个字符串。也就是在字符串中使用隔板分成2个单词。因为扫描字符串的时候会多次出现同一子字符串，所以采用动态规划。

f[i]表示扫描到i位置时是否符合条件。扫描数组元素i，从i倒序回退，查找点j，判断0到j和从j到i组成的单词是否符合条件

class Solution 
{
public:
    bool wordBreak(string s, vector<string>& wordDict) 
    {
        vector<bool> f(s.size() + 1, false);//长度为n的字符串有n+1个隔板
        f[0] = true; // 空字符串符合条件
        for (int i = 1; i <= s.size(); ++i) //扫描数组
            for (int j = i - 1; j >= 0; --j)
            {
                if (f[j] && find(wordDict.begin(),wordDict.end(),s.substr(j, i - j))!=wordDict.end())
                {
                    f[i] = true;
                    break;
                }
            }
        return f[s.size()];
    }
};

LeetCode 140. Word Break II

Given a non-empty string s and a dictionary wordDict containing a list of non-empty words, add spaces in s to construct a sentence where each word is a valid dictionary word. You may assume the dictionary does not contain duplicate words.

Return all such possible sentences.

For example, given.
s = “catsanddog”,
dict = [“cat”, “cats”, “and”, “sand”, “dog”].

A solution is [“cats and dog”, “cat sand dog”].

UPDATE (2017/1/4): The wordDict parameter had been changed to a list of strings (instead of a set of strings). Please reload the code definition to get the latest changes.

解析：不限制隔板的数量，需要用记忆化搜索来进行动态规划。扫描数组元素i，将其分为前后两部分，当后面的部分在字典中可以查找到，再递归分割前半部分。

class Solution {
public:
vector<string> wordBreak(string s, vector<string>& wordDict) {
    if(m.count(s)){//之前已经出现，不用再次计算
        return m[s];
    }
    vector<string> res;
    if(find(wordDict.begin(),wordDict.end(),s)!=wordDict.end())//此时s直接在字典中
        res.push_back(s);
    for(int i=1;i<s.size();i++){//以i为分界点切割
        string last=s.substr(i);//后面
        if(find(wordDict.begin(),wordDict.end(),last)!=wordDict.end()){//如果后面符合
            string pre=s.substr(0,i);//前面
            vector<string> tmp=combine(last,wordBreak(pre,wordDict));//后面的单词放入结果，继续dfs
            res.insert(res.end(),tmp.begin(),tmp.end());//将当前结果放入最终解
        }
    }
    m[s]=res;//记忆
    return res;
}
private:
vector<string> combine(string last,vector<string> pre){//处理结果，形成单词串
    for(int i=0;i<pre.size();i++){
        pre[i]+=" "+last;
    }
    return pre;
}
unordered_map<string,vector<string>> m;
};
