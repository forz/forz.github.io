LeetCode 78. Subsets

Given a set of distinct integers, nums, return all possible subsets.

Note: The solution set must not contain duplicate subsets.

For example,

If nums = [1,2,3], a solution is:

[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]

解析：要求输出所有子集，且输出有序，而每个元素只有出现和不出现之分，可以用0和1代替，直接用位操作暴力模拟。每个二进制位表示元素是否出现。

class Solution {
public:
vector<vector<int> > subsets(vector<int> &nums) {
    sort(nums.begin(), nums.end()); // 输出要求有序
    vector<vector<int> > result;
    const int n = nums.size();
    vector<int> v;
    for (int i = 0; i < 1 << n; i++) {//当前情况的二进制表示
        for (int j = 0; j < n; j++) {//将二进制转化为实际结果
            if (i & 1 << j) v.push_back(nums[j]);
        }
        result.push_back(v);
        v.clear();
    }
    return result;
}
};

LeetCode 90. Subsets II

Given a collection of integers that might contain duplicates, nums, return all possible subsets.

Note: The solution set must not contain duplicate subsets.

For example, If nums = [1,2,2], a solution is:

[  
  [2],  
  [1],  
  [1,2,2],  
  [2,2],  
  [1,2],  
  []  
]

解析：和上题的区别就在于出现了重复数字，枚举过程中会出现重复结果。使用set容器去重。

// 二进制法，时间复杂度O(2^n)，空间复杂度O(1)
class Solution {
public:
vector<vector<int> > subsetsWithDup(vector<int> &nums) {
    sort(nums.begin(), nums.end()); // 输出要求有序
    set<vector<int> > result; // 用 set 去重，不能用 unordered_set，因为输出要求有序
    int n = nums.size();
    vector<int> v;
    for (int i = 0; i < 1 << n; ++i) {//枚举所有情况
        for (int j = 0; j < n; ++j) {//输出情况
            if (i & 1 << j)//判断是否符合枚举情况
                v.push_back(nums[j]);
        }
        result.insert(v);//保存结果
        v.clear();
    }
    vector<vector<int> > real_result;//结果要求返回vector类型
    copy(result.begin(), result.end(), back_inserter(real_result));
    return real_result;
}
};

LeetCode 416. Partition Equal Subset Sum

Given a non-empty array containing only positive integers, find if the array can be partitioned into two subsets such that the sum of elements in both subsets is equal.

Note:

Each of the array element will not exceed 100.

The array size will not exceed 200.

Example 1:

Input: [1, 5, 11, 5]

Output: true

Explanation: The array can be partitioned as [1, 5, 5] and [11].

Example 2:

Input: [1, 2, 3, 5]

Output: false

Explanation: The array cannot be partitioned into equal sum subsets.

解析：

一个背包的题目，背包容量为数组中元素和的一半＋１，这样只要看是否有元素可以正好填满背包即可．

如果（当前位置－当前元素大小）位置可以通过放置之前的元素达到，则当前位置也可以通过放置当前 元素正好达到这个位置。状态转移方程为：

dp[i] = dp[i] || dp[i - nums[k]]; dp[i]表示能否分出一个和为i的集合

如果将 v 的循环顺序从逆序改成顺序的话，那么则成了 f[i][v]由 f[i][v-c[i]]推知，与本题意不符，

class Solution {  
public:  
bool canPartition(vector<int>& nums) {  
    int sum = accumulate(nums.begin(), nums.end(), 0);  //总和
    if(sum&1) return false;  //如果总和是奇数，不可能实现
    vector<int> dp(sum/2+1, 0);  
    dp[0] = 1;
    for(int i = 0; i < nums.size(); i++)//枚举物品
    {  
        for(int j = sum/2; j >= nums[i]; j--) //枚举背包容量（必须逆序）
            dp[j] = dp[j] || dp[j-nums[i]];  
    }  
    return dp[sum/2];  
}  
};
