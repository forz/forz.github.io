Flatten Binary Tree to Linked List

June 24, 2017

LeetCode 114. Flatten Binary Tree to Linked List#

Given a binary tree, flatten it to a linked list in-place.

For example, Given

    1
   / \
  2   5
 / \   \
3   4   6

The flattened tree should look like:

1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6

解析：

可以看出变换后的树实际上是按照先序遍历的顺序排列的，因此可以利用先序递归遍历，记录先序遍历的前驱结点，依次调整每个孩子的左右子树，这里需要注意的是遍历过程中需要先将当前节点的右子树记录下来，再调整当前节点的左右子树，然后递归调整左右子树。

class Solution {
public:
TreeNode *pre=NULL;
void flatten(TreeNode *root) {
    if(root==NULL)//递归终止条件
        return;
    if(pre)//调整父节点，将树变成链表形式
    {
        pre->left=NULL;
        pre->right=root;
    }
    pre=root;
    TreeNode *lastright=root->right;//记录当前节点的右子树，因为下一行递归过程中调整当前root节点，影响root的左右孩子。
    flatten(root->left);
    flatten(lastright);
}
};

补充：

递推方法。可以观察得到，形成链表后，左子树肯定在右子树前面，而且右子树紧跟在左子树的最右端孩子后面（前驱节点）。前序遍历中根节点的后继节点就是左孩子。

可以将每一个根节点的左子树移到右边，然后将原来的右子树挂在左子树的最右端孩子后面（前驱节点），然后根节点前进到其后继节点，也就是原来的左孩子。不断循环直到根节点为空。

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
 public:
 void flatten(TreeNode *root) {
    while(root != NULL){
        if(root->left != NULL){
            TreeNode *cur = root->left;//左子树的根节点
            while(cur->right!=NULL){//左子树的最右端孩子
                cur = cur->right;
            }
            cur->right = root->right;//将根节点的右子树挂在左子树最右端孩子下面暂时保存
            root->right = root->left;//调整链表
            root->left = NULL;
        }
        root = root->right;//前进到下一个节点，继续执行同样的操作
    }
}
};
