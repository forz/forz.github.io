LeetCode 96. Unique Binary Search Trees

Given n, how many structurally unique BST’s (binary search trees) that store values 1…n?

For example,
Given n = 3, there are a total of 5 unique BST’s.

1
2
3
4
5

1         3     3      2      1
 \       /     /      / \      \
  3     2     1      1   3      2
 /     /       \                 \
2     1         2                 3

G(n)：长度为n的序列的所有唯一的二叉树。F(i,n)，1<=i<=n：以i作为根节点的二叉树的数量。

给定一个序列1…..n，为了构造所有二叉树，我们可以使用1……n中的每一个数i作为根节点，自然1……(i-1)必然位于树的左子树中，(i+1)…..n位于树的右子树中。然后可以递归来构建左右子树，由于根节点是唯一的，所以可以保证构建的二叉树都是唯一的。

G(n) = G(0) G(n-1) + G(1) G(n-2) + … + G(n-1) G(0).

class Solution {  
public:  
int numTrees(int n) {  
    vector<int> G(n+1,0);
    //创建大小为n+1的int数组，并且进行初始化，默认为赋值0.
    //如果数组为空，毫无疑问，只有一种BST，即空树，f(0)=1。
    //如果数组仅有一个元素{1}，只有一种BST，单个节点，f(1)=1。
    G[0]=1;  
    G[1]=1;  
    for(int i=1;i<=n;i++)  
    {  
        for(int j=1;j<i+1;j++)  
            G[i]+=G[j-1]*G[i-j];  
    }  
    return G[n];  
}  
};

进阶：N的含义不变，假设可能的二叉树结构有M种，请返回M个二叉树的节点,每一棵二叉树代表一种可能的结构。

解析:

进阶问题与原问题的过程其实是很类似的。如果要生成中序遍历是{a…b}的所有结构，就从a开始一直到b,枚举每一个值作为头节点，把每次生成的二叉树结构的头节点都保存下来即可。假设其中一次是以i值为头节点的(a<=i<=b),以头节点的所有结构按如下步骤生成：

  1. 用{a…i-1}递归生成左子树的所有结构，假设所有结构的头节点保存在listLeft链表中。
  2. 用{a…i+1}递归生成右子树的所有结构，假设所有结构的头节点保存在listRight链表中。
  3. 在以i为头节点的前提下，listLeft中的每一种结构都可以与listRight中的每一种结构构成单独的结构，且和其他任何结构都不同。为了保证所有的结构之间不互相交叉，所以对每一种结构都复制出新的树，并记录在总的链表res中。

	public static List<Node> generate(int start, int end) {
		List<Node> res = new LinkedList<Node>();
		if (start > end) {
			res.add(null);
		}
		Node head = null;
		for (int i = start; i < end + 1; i++) {
			head = new Node(i);
			List<Node> lSubs = generate(start, i - 1);
			List<Node> rSubs = generate(i + 1, end);
			for (Node l : lSubs) {
				for (Node r : rSubs) {
					head.left = l;
					head.right = r;
					res.add(cloneTree(head));
				}
			}
		}
		return res;
	}

	public static Node cloneTree(Node head) {
		if (head == null) {
			return null;
		}
		Node res = new Node(head.value);
		res.left = cloneTree(head.left);
		res.right = cloneTree(head.right);
		return res;
	}

LeetCode 95. Unique Binary Search Trees II

Given an integer n, generate all structurally unique BST’s (binary search trees) that store values 1…n.

For example,
Given n = 3, your program should return all 5 unique BST’s shown below.

3
4
5

1         3     3      2      1
 \       /     /      / \      \
  3     2     1      1   3      2
 /     /       \                 \
2     1         2                 3

解析：要求返回所有的BST,用dfs求解，BST的种类取决于其最大值和最小值。每次dfs返回以该区间构成的所有子树的根节点。

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    vector<TreeNode *> generateTree(int from, int to)
    {
        vector<TreeNode *> ret;
        //ret保存当前树的所有节点情况（节点重复，但每个节点都有独特的两个孩子）
        if(to - from < 0) ret.push_back(0);//没有节点，放入NULL
        if(to - from == 0) ret.push_back(new TreeNode(from));//有一个节点，直接放入
        if(to - from > 0)//有多个节点
        {
            for(int i=from; i<=to; i++)//将节点划分左右子树（固定根节点，左右子树的范围确定）
            {
                vector<TreeNode *> l = generateTree(from, i-1);//递归求子树
                vector<TreeNode *> r = generateTree(i+1, to);
                for(int j=0; j<l.size(); j++)//枚举子树节点（每个节点都可以作为子树的根节点）
                {
                    for(int k=0; k<r.size(); k++)
                    {
                        TreeNode * h = new TreeNode (i);
                        h->left = l[j];
                        h->right = r[k];
                        ret.push_back(h);
                    }
                }
            }
        }
        return ret;
    }
    vector<TreeNode *> generateTrees(int n) {
        if(n==0) //如果大小是0，直接返回空
            return (vector<TreeNode *>)0;
        return generateTree(1, n);
    }
};
