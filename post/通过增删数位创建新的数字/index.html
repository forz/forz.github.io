LeetCode 402. Remove K Digits

Given a non-negative integer num represented as a string, remove k digits from the number so that the new number is the smallest possible.

Note:

The length of num is less than 10002 and will be ≥ k.

The given num does not contain any leading zero.

Example 1:

Input: num = "1432219", k = 3  
Output: "1219"

Explanation: Remove the three digits 4, 3, and 2 to form the new number 1219 which is the smallest.

Example 2:

Input: num = "10200", k = 1  
Output: "200"

Explanation: Remove the leading 1 and the number is 200. Note that the output must not contain leading zeroes.

Example 3:

Input: num = "10", k = 2
Output: "0"

Explanation: Remove all the digits from the number and it is left with nothing which is 0.

解析：

要求最小子序列，需要用到单调栈(递增栈）,可删除的数量k已知，如果单调栈弹出的元素数目超过可删除的数量k，那么单调栈就不能继续弹出元素。这是固定弹出元素个数的单调栈。

而且在单调栈中，若想找到最小子序列，肯定要删除最多的元素。可以贪心得到最后组成的数的总长度就是len-k。所以结果直接取单调栈的前len-k个元素，此时前面的元素包括前缀0，需要去掉。

class Solution {
public:
string removeKdigits(string num, int k) {
    string ans;//string也可以当作栈使用
    int n = k, len = num.size(), cnt = 0;//n代表可删除的元素数
    for(auto val: num)//扫描字符
    {
        while(!ans.empty() && n > 0 && val < ans.back())
//如果当前字符比栈顶元素小，并且还可以继续删除元素，那么就将栈顶元素删掉，
//这样可以保证将当前元素加进去一定可以得到一个较小的序列．   
        {
            n--;
            ans.pop_back();
        }
        ans.push_back(val);
    }
    while(ans[cnt]=='0') cnt++;//字符串首字符不为０
//最后我们只取前len-k个元素构成一个序列即可，如果这样得到的是一个空串那就手动返回０
    ans = ans.substr(cnt, len-k-cnt);
    return !ans.size()?"0":ans;
}
};

LeetCode 321. Create Maximum Number

Given two arrays of length m and n with digits 0-9 representing two numbers. Create the maximum number of length k <= m + n from digits of the two. The relative order of the digits from the same array must be preserved. Return an array of the k digits. You should try to optimize your time and space complexity.

Example 1:

nums1 = [3, 4, 6, 5]   
nums2 = [9, 1, 2, 5, 8, 3]  
k = 5  
return [9, 8, 6, 5, 3]

Example 2:

nums1 = [6, 7]  
nums2 = [6, 0, 4]  
k = 5  
return [6, 7, 6, 0, 4]

Example 3:

nums1 = [3, 9]  
nums2 = [8, 9]  
k = 3  
return [9, 8, 9]

解析：

我们首先把k个数分成两部分，i 和 k-i,

由于k的大小不定，所以有三种可能：

第一种是当k为0时，两个数组中都不取数；

第二种是当k不大于其中一个数组的长度时，有可能只从一个数组中取 数；

第三种情况是k大于其中一个数组的长度，则需要从两个数组中分别取数，

接下来要从每个数组中挑选t个数字组成最大的子序列，用递减栈处理，处理过程中要保证栈的容量能到达K。这是固定容量的单调栈。需要保证弹出元素的过程中剩余元素大于容量，并且当栈满时不能再入栈。

最后是将两个子序列合并，涉及比较问题。此处要模拟一个大数比较算法。

如果两个数的某一位不相等，那么取较大值，如果两个数相等需要一直往后比，直到它们不相等分出大小为止。如果有一方比较过程中已经遍历结束，那么按这方较小来算。

class Solution {
public:
vector<int> maxNumber(vector<int>& nums1, vector<int>& nums2, int k) {
int m = nums1.size();
int n = nums2.size();
vector<int> result(k);
for (int i = max(0 , k - n); i <= k && i <= m; i++){

    auto v1 = maxArray(nums1, i);
    auto v2 = maxArray(nums2, k - i);
    vector<int> candidate = merge(v1, v2, k);
    //更新最优解也可以使用大数比较算法。
    if (greater(candidate, 0, result, 0)) result = candidate;
}
return result;
}

vector<int> maxArray(vector<int>& nums, int k){

int n = nums.size();
vector<int> result(k);//新建一个空栈stack，j为栈顶元素的下标
for (int i = 0; i < n; i++){
    while ( !result.empty() && n - i + result.size()>k && result.back() < nums[i]) 
        result.pop_back();
   // 要保证剩余的数组足以填满栈至容量k
    if (result.size()< k) result.push_back(nums[i]);//如果栈的容量小于k，则压入nums[i]
}
return result;
}
vector<int> merge(vector<int>& nums1, vector<int>& nums2, int k) {
//我们要做k次选择，得到 k = m+n的最大数
vector<int> ans(k);
int i = 0, j = 0;
for (int r = 0; r<k; r++){
    if( greater(nums1, i, nums2, j) ) ans[r] = nums1[i++] ;
    else ans[r] = nums2[j++];
}
return ans;
}
bool greater(vector<int>& nums1, int i, vector<int>& nums2, int j){//判断nums1元素是否比nums2的元素大
while (i < nums1.size() && j < nums2.size() && nums1[i] == nums2[j]){
    i++;
    j++;
}
    //返回2种为真的情况。
return j == nums2.size() || (i<nums1.size() && nums1[i] > nums2[j]);
}
};
