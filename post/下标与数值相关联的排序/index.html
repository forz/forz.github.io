下标与数值相关联的排序

2017-09-01
解题报告
约 768 字 预计阅读 2 分钟

文章目录

题目:

给定一个长度为N的整型数组arr,其中有N个互不相等的自然数1〜N,请实现arr的排序，但是不要把下标0〜N-1位置上的数通过直接赋值的方式替换成1〜N。

要求:

时间复杂度为O(N),额外空间复杂度为0(1)。

解析:

arr在调整之后应该是下标从0到AM的位置上依次放着1〜N即arr[index]=index+l。

  1. 从左到右遍历am假设当前遍历到/位置。
  2. 如果arr[i]==i+l,说明当前的位置不需要调整，继续遍历下一个位置。
  3. 如果arr[i]!=i+l，说明此时/位置的数arr[i]不应该放在/位置上，接下来将进行跳 的过程。

举例来说明，比如[1，2,5,3,4],假设遍历到位置2,也就是5这个数。5应该放在位置4上，所以把5放过去，数组变成[1，2,5,3,5]。同时，4这个数是被5替下来的数，应该放在位置3,所以把4放过去，数组变成[丨，2,5,4,5]。同时3这个数是被4替下来的数，应该放在位置2,所以把3放过去，数组变成[1，2,3,4,5]。当跳了一圈回到原位置后，会发现此时arr[i]=i+1，继续遍历下一个位置。

	public static void sort1(int[] arr) {
		int tmp = 0;
		int next = 0;
		for (int i = 0; i != arr.length; i++) {
			tmp = arr[i];
			while (arr[i] != i + 1) {
				next = arr[tmp - 1];
				arr[tmp - 1] = tmp;
				tmp = next;
			}
		}
	}

	public static void modify(int[] arr) {
		if (arr == null || arr.length < 2) {
			return;
		}
		int even = 0;
		int odd = 1;
		int end = arr.length - 1;
		while (even <= end && odd <= end) {
			printArray(arr);
			if ((arr[end] & 1) == 0) {
				swap(arr, end, even);
				even += 2;
			} else {
				swap(arr, end, odd);
				odd += 2;
			}
		}
	}

	public static void swap(int[] arr, int index1, int index2) {
		int tmp = arr[index1];
		arr[index1] = arr[index2];
		arr[index2] = tmp;
	}
