求两个数组的交集

June 25, 2017

有序数组#

以下两种方法都可以适用于数组中数据重复的情况。

长度相似#

采用二路归并法来进行求解交集

对于数组A,B分别以i,j从头遍历数组。如果当前位置的A[i]等于B[j]，则这两个数是两个数组的一个交集，记录下来并继续遍历；如果A[i]大于B[j]，则继续遍历数组B，否则遍历数组A。

int comNum(int a[],int m,int b[],int n,int *p)  
{  
    //m,n分别为整型数组a,b的长度            
    int i=0,j=0,cnt=0;  
    while(i<m&&j<n)  
    {  
        if(a[i]==b[j])  
        {  
            p[cnt++]=a[i];  
            i++;  
            j++;  
        }  
        else if(a[i]>b[j])  
            j++;  
        else  
            i++;  
    }  
    return cnt;  
}

长度悬殊#

二分查找法

枚举长度较小的数组，然后将这个元素值在在长度较大的数组中进行二分查找。

int ComNum_Binary(int a[],int lenA,int b[],int lenB,int *comlst)
{
    int i,j,k=0;
    for(i = 0; i < lenA; i++)//数组A长度小
    {
        int l=0,r=lenB-1,mid;
        while(l <= r)
        {
            mid = r+(l-r)/2;
            if( a[i] < b[mid] )
                r = mid -1;
            else if( a[i] > b[mid] )
                l = mid +1;
            else
            {
                comlst[k++] = a[i];
                break;
            }
        }
    }
    return k;
}

无序数组#

无序数组无需考虑数组大小，使用hash方法来求解。

hash方法要考虑的问题在于是否允许重复元素，根据这个条件来选择合适的hash容器。

  1. 如果出现重复元素，就用unordered_map，因为每个map中可以记录元素出现的次数。

  2. 如果不出现重复元素，就用unordered_set。
