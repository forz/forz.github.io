LeetCode 413. Arithmetic Slices

Given an m x n matrix of non-negative integers representing the height of each unit cell in a continent, the “Pacific ocean” touches the left and top edges of the matrix and the “Atlantic ocean” touches the right and bottom edges.

Water can only flow in four directions (up, down, left, or right) from a cell to another one with height equal or lower.

Find the list of grid coordinates where water can flow to both the Pacific and Atlantic ocean.

Note:

The order of returned grid coordinates does not matter.

Both m and n are less than 150.

Example:

Given the following 5x5 matrix:

Pacific ~   ~   ~   ~   ~ 
     ~  1   2   2   3  (5) *
     ~  3   2   3  (4) (4) *
     ~  2   4  (5)  3   1  *
     ~ (6) (7)  1   4   5  *
     ~ (5)  1   1   2   4  *
        *   *   *   *   * Atlantic

Return:

[[0, 4], [1, 3], [1, 4], [2, 2], [3, 0], [3, 1], [4, 0]] (positions with parentheses in above matrix).

解析：

题目给了一个矩阵，代表一个高度，高度高的可以流到同高度或更小的高度那里

而在矩阵的四周，围绕着太平洋和大西洋（可以看图示，太平洋左边和上边，大西洋下边和右边，也就是这四个边对应的位置是通向海洋的）

现在问说这个矩阵里面的位置，那些的水流可以同时流到大西洋和太平洋？

分别处理每个海洋，从海洋边缘（每个海洋两条边）开始，一步步的搜索，即从连接海洋的地方还是搜索，哪些节点的高度高于等于自身（反过来就是可以从那里流到自己），如果是，就标记为true，就这样不断搜索下去。最后所有标记为true的位置就是可以流到对应的海洋。

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
public:
vector<pair<int, int>> res;
vector<vector<int>> visited;
//利用位操作保存当前结点所连海域（一个数组当两个数组用）
void dfs(vector<vector<int>>& matrix, int x, int y, int pre, int preval){
    //递归过程中有2个量需要保存：前驱节点pre和相连海域preval
    if (x < 0 || x >= matrix.size() || y < 0 || y >= matrix[0].size()  
            || matrix[x][y] < pre || (visited[x][y] & preval) == preval)
        return;            //保证可以流到当前点，并且该点未与前驱节点所连的海域相同（保证不重复）
    visited[x][y] |= preval;//维护新的值
    if (visited[x][y] == 3) res.push_back({x, y});
    //0011 表示两个海域相连，放入结果
    dfs(matrix, x + 1, y, matrix[x][y], visited[x][y]); //四个方向
    dfs(matrix, x - 1, y, matrix[x][y], visited[x][y]);
    dfs(matrix, x, y + 1, matrix[x][y], visited[x][y]); 
    dfs(matrix, x, y - 1, matrix[x][y], visited[x][y]);
}
vector<pair<int, int>> pacificAtlantic(vector<vector<int>>& matrix) {
    //因为最终结果是水流到海里面，所以直接从海岸开始往中间找比海岸高的点更好
    if (matrix.empty()) return res;
    int m = matrix.size(), n = matrix[0].size();//读取行和列
    visited.resize(m, vector<int>(n, 0));//重新分配visited大小（）
    for (int i = 0; i < m; i++) {//左右边界（初始的前驱节点（大海）的matrix值设为min）
        dfs(matrix, i, 0, INT_MIN, 1);//1代表太平洋
        dfs(matrix, i, n - 1, INT_MIN, 2);//2代表大西洋
    }
    for (int i = 0; i < n; i++) {//上下边界
        dfs(matrix, 0, i, INT_MIN, 1);
        dfs(matrix, m - 1, i, INT_MIN, 2);
    }
    return res;
}
};
