LeetCode 9. Palindrome Number#

Determine whether an integer is a palindrome. Do this without extra space.

Some hints:
Could negative integers be palindromes? (ie, -1)

If you are thinking of converting the integer to string, note the restriction of using extra space.

You could also try reversing an integer. However, if you have solved the problem “Reverse Integer”, you know that the reversed integer might overflow. How would you handle such case?

解析：负数不是回文数，题目要求只使用O(1)的空间，所以只设置一个局部变量sum，为了防止越界，只拿sum和x的一半去比较。若x是奇数位，则sum/10==x，若x为偶数位，则sum==x

class Solution {
public:
bool isPalindrome(int x) {
    if(x<0|| (x!=0 &&x%10==0)) return false;//排除负数和单个数
    int sum=0;
    while(x>sum)
    {
        sum = sum*10+x%10;
        x = x/10;
    }
    return (x==sum)||(x==sum/10);
}
};

LeetCode 125. Valid Palindrome#

Given a string, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.

For example, “A man, a plan, a canal: Panama” is a palindrome. “race a car” is not a palindrome.

解析：

如果只是单纯的判断一个字符串是否为回文串，双指针逼近是最简单的。

isalpha如果是字母，返回一个非零数；否则返回为0 isalnum如果是字母或数字，返回一个非零数；否则返回为0 isdigit如果是数字（0-9）返回一个非零数；否则返回为0

class Solution {
public:
bool isPalindrome(string s) {
for (int i = 0, j = s.size() - 1; i < j; i++, j--) { //双指针
    while (isalnum(s[i]) == false && i < j) i++; //找到需比较的字符
    while (isalnum(s[j]) == false && i < j) j--;//找到需比较的字符
    if (toupper(s[i]) != toupper(s[j])) return false; //统一大小写
}
return true;
}
};

LeetCode 234. Palindrome Linked List#

Given a singly linked list, determine if it is a palindrome.

Follow up: Could you do it in O(n) time and O(1) space?

解析：因为本题需要O(1)的空间限制，只能在原链表中修改。先用快慢指针法求出链表的中心，然后将后半部分翻转，再用前半部分和后半部分比较即可。

class Solution {
public:
bool isPalindrome(ListNode* head) {
    //如果链表为空或者仅有一个元素那么肯定是回文链表
    if (!head || !head->next) return true;
    //快慢指针法，寻找链表中心 
    ListNode *slow = head, *fast = head;
    while (fast->next && fast->next->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    ListNode * pre=NULL;  
    ListNode * cur=slow->next;  
    ListNode * tmp=NULL;  
    while(cur)  
    {  
        tmp=cur->next;  
        cur->next=pre;  
        pre=cur;  
        cur=tmp;  
    }  
    while (pre) {//进行回文比较
        if (head->val != pre->val) return false;
        pre = pre->next;
        head = head->next;
    }
    return true;
}
};
