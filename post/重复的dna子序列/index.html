重复的DNA子序列

June 24, 2017

LeetCode 187. Repeated DNA Sequences#

All DNA is composed of a series of nucleotides abbreviated as A, C, G, and T, for example: “ACGAATTCCG”. When studying DNA, it is sometimes useful to identify repeated sequences within the DNA.

Write a function to find all the 10-letter-long sequences (substrings) that occur more than once in a DNA molecule.

For example,

Given s = “AAAAACCCCCAAAAACCCCCCAAAAAGGGTTT”,

Return:

[“AAAAACCCCC”, “CCCCCAAAAA”].

解析：

要找到在一个字符串中多次重复出现的单词，一般情况下要使用hash表，将单词字典存入，然后枚举字符串的起点和终点来进行扫描。

本题没有给出单词字典，但是因为每个位置只能出现4个字符，长度固定为10，可以发现输入的可能性是可以计算出来的，即4^10个。因为每个位置上只有4种字符，可以将单词哈希成int值放入map，一共4种字符，每个字符用2个二进制位来表示。每个数字占2位，num的最大值为4^16。这样节约了算具体单词的时间，可以直接处理字符串。

将ACGT映射成1，2，3，4。A=1，C=3，G=7，T=20，对5取余可以让它们区分开。

扫描字符串的时候需要维护窗口大小，在二进制下有特殊的维护方式。

不需要删除最前面的一个字符来维护窗口大小，直接左移num即可。

0xfffff代表了20个1组成的二进制序列，按位与之后，结果就是当前字符串代表的数值（取其低20位）

class Solution {
public:
vector<string> findRepeatedDnaSequences(string s) {
    char map[4<<10] = {0};
    int len = s.length(), num = 0;
    vector<string> res;
    if (len <= 10) {
        return res;
    }
    for (int i = 0; i < 9; ++i) {//先把字符串前9个放入num中
        num <<= 2;
        num |= (s[i] - 'A' + 1) % 5;
    }
    for (int i = 9; i < s.length(); ++i) {
    //每放入一个字符，维护一次结果
        num <<= 2;        
        num |= (s[i] - 'A' + 1) % 5;
        num = num & 0xfffff;
        if (map[num]++ == 1) {
        //判断是否重复2次，为了防止结果出现重复，只判断出现1次的，大于1次说明之前肯定放入结果了
            if(find(res.begin(),res.end(),s.substr(i - 9, 10))==res.end())
            res.push_back(s.substr(i - 9, 10));
        }
    }
    return res;
}
};
