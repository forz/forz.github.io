LeetCode 115. Distinct Subsequences#

Given a string S and a string T, count the number of distinct subsequences of T in S.

A subsequence of a string is a new string which is formed from the original string by deleting some (can be none) of the characters without disturbing the relative positions of the remaining characters. (ie, “ACE” is a subsequence of “ABCDE” while “AEC” is not).

Here is an example:

S = “rabbbit”, T = “rabbit”

Return 3.

解析：

定义二维数组dp[i][j]为字符串 s前i个字符 变换到 t前j个字符 的变换方法（0有特殊的意义，表示为空，所以不能从0开始计数）

dp[0][0] = 1，dp[i][0] = 0（把任意一个字符串变换为一个空串只有一个方法）

如果S[i-1]==T[j-1]，那么dp[i][j] = dp[i-1][j-1] + dp[i-1][j]。 意思是如果当前S[i-1]==T[j-1]，那么当前这个字母即可以保留也可以抛弃，所以变换方法等于保留这个字母的变换方法加上不用这个字母的变换方法。

如果S[i]!=T[i-1]，那么dp[i][j] = dp[i-1][j]， 意思是如果当前字符不等，那么就只能抛弃当前这个字符。

class Solution {
public:
int numDistinct(string s, string t) {
    if (s.empty() || t.empty()||s.length() < t.length())
        return 0;
    else
    {
        int ls = s.length(), lt = t.length();
        vector<vector<int> > dp(ls + 1, vector<int>(lt + 1, 0));
        dp[0][0] = 1;
        for (int i = 0; i < ls; ++i)
            dp[i][0] = 1;
        for (int i = 1; i <= ls; ++i)
        {
            for (int j = 1; j <= lt; ++j)
            {
                dp[i][j] = dp[i - 1][j];/*首先不管当前字符是否相同，为dp[i][j]赋初值*/
                if (s[i-1] == t[j-1])
                    /*如果s和t的当前字符相同，有两种选择保留或不保留*/
                    dp[i][j] += dp[i - 1][j - 1];
            }
        }
        return dp[ls][lt];
    }
}    
};

补充：

可以看出，此题可以用滚动数组来优化。注意此题每个字符只能选择用或不用，而且只能用一次，属于01背包问题。如果用普通的直接降第一维的处理方法，会出现该字符多次重复使用的问题。所以第二维要逆序遍历，保证每一次利用的之前的递推值都是不使用该字符的结果。

f[i][v]=max{f[i-1][v],f[i-1][v-c[i]]+w[i]} ==> f[v]=max{f[v],f[v-cost]+weight}

在每次主循环中我们以 v=V..0 的顺序推 f[v]，这样能保证推f[v]时 f[v-c[i]]保存的是状态 f[i-1][v-c[i]]的值。而且dp条件按照顺序枚举时的i和j判断即可。

class Solution {
public:
int numDistinct(const string &S, const string &T) {
    //dfs可以做，但超时的时候考虑动态规划
    vector<int> f(T.size() + 1);
    f[0] = 1;
    for (int i = 0; i < S.size(); ++i) {//扫描串S
        for (int j = T.size() - 1; j >= 0; --j) {//逆序枚举串T
            f[j + 1] += S[i] == T[j] ? f[j] : 0;
        }
    }

    return f[T.size()];
}
};

LeetCode 72. Edit Distance#

Given two words word1 and word2, find the minimum number of steps required to convert word1 to word2. (each operation is counted as 1 step.)

You have the following 3 operations permitted on a word:

a) Insert a character.
b) Delete a character.
c) Replace a character.

解析:

dp[i][j]表示第一个字符串前i个字符转换到第二个字符串前j个字符需要的步数（前闭后开）

s1[i] 和 s2[j] 如果相等，就不需要进行操作

dp[i][j] = dp[i-1][j-1];

s1[i] 和 s2[j] 如果不相等，需要进行操作

(在s1中前i个字符中找到方法转换成s2中前j个字符，每个dp值代表一种方法)

insert: dp[i][j] = dp[i][j - 1] + 1;

执行完DP(i, j-1)的操作，最后一步是在i后面增加j字符

replace: dp[i][j] = dp[i - 1][j - 1] + 1;

执行完DP(i-1, j-1)的操作，最后一步是将第i个字符替换成j

delete: dp[i][j] = dp[i - 1][j] + 1;

执行完DP(i-1, j)的操作，最后一步是删除第i个字符

class Solution {
public:
int minDistance(string word1, string word2) {
    int m =word1.size(), n =word2.size();
    int vector<vecotr<int>>dp(m+1,vector<int>(n+1)); 
    dp[0][0]=0;//初始化边界值
    for (int i=1; i<n+1; i++)   dp[0][i] = i;//全靠增加
    for (int i=1; i<m+1; i++)   dp[i][0] = i;//全靠减少
    for (int i=1; i<=m; i++) 
        for (int j=1; j<=n; j++) {
            if (word1[i-1] == word2[j-1]) 
                dp[i][j] = dp[i-1][j-1];//相等就不需要增加步数
            else //不相等可以通过修改，删除，增加来求得步数
                dp[i][j] = min(dp[i-1][j-1], min(dp[i-1][j], dp[i][j-1])) + 1;
        }
    return dp[m][n];
}
};

补充：滚动数组优化，优化虽然减少一维，但所有初始化dp值和递推公式都不能少。

f[j]表示转换到第2个字符串前j个字符需要的步数，也就是保留第二维，删去第一维。

因为初始化dp值的时候要初始化第一维dp[i][0]=i，这个不能因为删去第一维就省略，可以将这个放在第一重循环（循环第一维）和第二重循环（循环第二维）之间。

class Solution {
public:
int minDistance(const string &word1, const string &word2) {
    if (word1.length() < word2.length())
        return minDistance(word2, word1);
    //默认word1更长，两者互换不影响结果
    int f[word2.length() + 1];//选取较短的长度，节约空间
    int upper_left = 0; // 额外用一个变量记录dp[i-1][j-1],因为dp需要三个之前的dp值
    for (int i = 0; i <= word2.size(); ++i)//全靠在空串中增加字符来构成第二个字符串
        f[i] = i;            //dp[0][i] = i;
    for (int i = 1; i <= word1.size(); ++i) {//扫描s1串的字符
        upper_left = f[0];
        f[0] = i;        //dp[i][0] = i;  更新下次循环时的upper_left
        for (int j = 1; j <= word2.size(); ++j) {
            int upper = f[j];    //暂存f[j]
            if (word1[i - 1] == word2[j - 1])
                f[j] = upper_left;
            else
                f[j] = 1 + min(upper_left, min(f[j], f[j - 1]));
                //dp[i][j] = min(dp[i-1][j-1], min(dp[i-1][j], dp[i][j-1])) + 1;
            upper_left = upper;//保存f[j-1]
        }
    }
    return f[word2.length()];
}
};
