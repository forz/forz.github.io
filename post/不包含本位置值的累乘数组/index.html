LeetCode 238. Product of Array Except Self

Given an array of n integers where n > 1, nums, return an array output such that output[i] is equal to the product of all the elements of nums except nums[i].

Solve it without division and in O(n).

For example, given [1,2,3,4], return [24,12,8,6].

Follow up:

Could you solve it with constant space complexity? (Note: The output array does not count as extra space for the purpose of space complexity analysis.)

解析：在result中保存left值。right值用变量代替.先正序扫一遍，计算出每个元素左端的乘积，直接放在result中。再倒序扫一遍，用right保存每个元素右端的乘积，边计算，边维护right值。

class Solution {
public:
vector<int> productExceptSelf(vector<int>& nums) {
    vector<int> res(nums.size(), 1);//res数组保存结果
    for (int i = 1; i < nums.size(); ++i) //正序遍历
        res[i] = res[i - 1] * nums[i - 1];//res保存i左面的乘积
    int right = 1;//right不需要建立数组，直接在res中记录
    for (int i = nums.size() - 1; i >= 0; --i) {//逆序遍历
        res[i] *= right;//res保存i的左面和右面所有数的乘积
        right *= nums[i];//更新right
    }
    return res;
}
};
