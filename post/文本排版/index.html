LeetCode 68. Text Justification

Given an array of words and a length L, format the text such that each line has exactly L characters and is fully (left and right) justified.

You should pack your words in a greedy approach; that is, pack as many words as you can in each line. Pad extra spaces ‘ ‘ when necessary so that each line has exactly L characters.

Extra spaces between words should be distributed as evenly as possible. If the number of spaces on a line do not divide evenly between words, the empty slots on the left will be assigned more spaces than the slots on the right.

For the last line of text, it should be left justified and no extra space is inserted between words.

For example,

words: [“This”, “is”, “an”, “example”, “of”, “text”, “justification.”]

L: 16.

Return the formatted lines as:

[
   "This    is    an",
   "example  of text",
   "justification.  "
]

解析：将多个单词按照L长度分行。先计算出每行最多可以放的单词数k和单词总长度l，然后在每一行分配空格。吐过是末行，所有单词间放一个空格，如果不是末行，需要进行计算

class Solution {
public:
vector<string> fullJustify(vector<string> &words, int L) {
vector<string> res;
for(int i = 0, k, l; i < words.size(); i += k) {//扫描每个字符串i
    for(k = l = 0; i + k < words.size() && l + words[i+k].size() <= L - k; k++) 
    //l为当前行已使用长度 k为当前行能容纳的字符串，因为要区别每个字符串，所以行内要预留k个位置来放空格（k从0开始计数），所以实际可用长度为L-k
        l += words[i+k].size();
    string tmp = words[i];//先将第一个单词放入
    for(int j = 0; j < k - 1; j++) {//处理当前行能容纳的所有字符串
        if(i + k >= words.size()) tmp += " ";//末行时所有单词间放一个空格
        else tmp += string((L - l) / (k - 1) + (j < (L - l) % (k - 1)), ' ');
        //L-l为总空格数 k-1为单词数（之前for循环最后还进行了自加）  j为当前字符串在该行内的序号
//非末行时：先算出平均空格数，余数空格按照单词前后顺序，靠前的增加1个空格，靠后的没有。
        tmp += words[i+j+1];//从第二个单词开始填
    }
    tmp += string(L - tmp.size(), ' ');//末行情况下还要补空格
    res.push_back(tmp);//放入结果
}
return res;
}
};
