数组排序之后相邻数的最大差值

June 24, 2017

LeetCode 164. Maximum Gap#

Given an unsorted array, find the maximum difference between the successive elements in its sorted form.

Try to solve it in linear time/space.

Return 0 if the array contains less than 2 elements.

You may assume all elements in the array are non-negative integers and fit in the 32-bit signed integer range.

解析：

桶排序要求所有元素在一定范围内。按照元素范围创建桶，每个元素映射到桶中最后扫描桶，每个桶有几个元素，就相当于原数组有几个元素，类似于hash。时间复杂度是O(n)，空间复杂度是O(n),拿空间换时间。

给定一个未排序的数组，找到其排序形式的连续元素之间的最大差异，在O(n)条件下只能靠桶排序和基数排序。连续元素之间的最大差异就相当于求桶的最大差异（要保证桶足够小，让最大差异发生在桶之间）

假设输入的数组中的个数是n,范围从a~b。则我们会发先最大的gap一定会大于等于ceiling(b-a)/(n-1),因为如果数组是等差数列，数与数之间的差异是ceiling(b-a)/(n-1)，所以最大gap不可能比这个还小。所以我们假设让桶的容量变成ceiling(b-a)/(n-1)，那最大差异一定发生的桶之间。

由于同一个桶内的元素之间的差值至多为range-1，因此最终答案不会从同一个桶中选择。

先求出数组的最大值和最小值，确定建立桶的范围。然后扫描数组，维护每个桶的最大值和最小值。

对于每一个非空的桶p，找出下一个非空的桶q，则q.min - p.max可能就是备选答案。返回所有这些可能值中的最大值。

class Solution {
public:
    int maximumGap(vector<int>& nums) {
        const int size_num = nums.size();
        if (size_num < 2) return 0;
        int maxV = *max_element(nums.begin(), nums.end());//最大值
        int minV = *min_element(nums.begin(), nums.end());//最小值
        if (maxV == minV) return 0;
        double range = (maxV - minV) / double(size_num - 1);
        vector<int> max_b(size_num, INT_MIN), min_b(size_num, INT_MAX);
         // 每个桶都维护一个最大值和最小值
        for (int i = 0; i < size_num; i++) {//扫描数组
            int index = (nums[i] - minV) / range;//将数组元素映射到桶后的新下标
            max_b[index] = max(max_b[index], nums[i]);
            min_b[index] = min(min_b[index], nums[i]);
        }
        int max_g = (int)range,  start = max_b[0];//存储第一个桶的最大值
        for (int i = 1; i < size_num; i++) {
            if (min_b[i] == INT_MAX) continue;//空桶，跳过
            max_g = max(max_g, min_b[i] - start);//下一个桶的最小值也比上一个桶的最大值大
            start = max_b[i];
        }
        return max_g;
    }
};
