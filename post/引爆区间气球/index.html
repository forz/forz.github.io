引爆区间气球

June 24, 2017

LeetCode 312. Burst Balloons#

Given n balloons, indexed from 0 to n-1. Each balloon is painted with a number on it represented by array nums. You are asked to burst all the balloons. If the you burst balloon i you will get nums[left] nums[i] nums[right] coins. Here left and right are adjacent indices of i. After the burst, the left and right then becomes adjacent.

Find the maximum coins you can collect by bursting the balloons wisely.

Note:

(1) You may imagine nums[-1] = nums[n] = 1. They are not real therefore you can not burst them.

(2) 0 ≤ n ≤ 500, 0 ≤ nums[i] ≤ 100

Example:

Given [3, 1, 5, 8]

Return 167

nums = [3,1,5,8] --> [3,5,8] -->   [3,8]   -->  [8]  --> []
coins =  3*1*5      +  3*5*8    +  1*3*8      + 1*8*1   = 167

解析：

结果与区间有关系，所以用区间DP

首先，一排气球左边是i，右边是j，我们可以假设最后一个打爆的气球是k，i到j这段区间所能得到的最大得分为dp[i][j]。

状态转移方程为dp[i][j] = max(i < k < j) (dp[i][k] + dp[k][j] + a[i] a[k] a[j])

nums[i] nums[j] nums[k]表示最后一次得分，这里我们已经假设i到k之间，k到j之间气球已经都打爆了。

所以要加上dp[i][k]+dp[k][j]，其实就是最后一个气球之前的气球。

先枚举区间大小，再枚举左边界，保证先算小区间的dp值

class Solution {
public:
int maxCoins(vector<int>& nums) {
    nums.insert(nums.begin(),1);
    nums.push_back(1);
    int n=nums.size();
    vector<vector<int>> dp(n,vector<int>(n,0));//dp初始值全为0，因为找最大值
    for(int i=2;i<n;i++)//区间大小
    {
        for(int left=0;left<n-i;left++)//左边界
        {
            int right=left+i;
            for(int j=left+1;j<right;j++)//扫描子区间
                dp[left][right]=max(dp[left][right],dp[left][j]+dp[j][right]+nums[left]*nums[right]*nums[j]);
        }
    }
    return dp[0][n-1];
}
};
