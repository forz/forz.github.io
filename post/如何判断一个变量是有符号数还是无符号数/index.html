如何判断一个变量是有符号数还是无符号数

2017-09-05
解题报告
约 121 字 预计阅读 1 分钟

文章目录

通过改变符号位判断。把A进行一个位运算，将最高位置1,判断是否大于0。

程序示例如下：

#include <stdio.h>
int main()
{
    unsigned A = 10;
    A = A|(1<<31);
    if(A > 0)
        printf("无符号数\n");
    else
        printf("有符号数\n");
    return 0;
}
