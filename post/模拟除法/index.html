    LeetCode 166. Fraction to Recurring Decimal

    Given two integers representing the numerator and denominator of a fraction, return the fraction in string format.

    If the fractional part is repeating, enclose the repeating part in parentheses.

    For example,

    Given numerator = 1, denominator = 2, return “0.5”.

    Given numerator = 2, denominator = 1, return “2”.

    Given numerator = 2, denominator = 3, return “0.(6)”.

    解析：

    可能出现INT_MIN/-1的越界情况，因此第一步现将int转为long long int

    INT_MIN转化为正数会溢出，在64位的操作系统，所以INT的范围是：-2147483648 ~ 2147483647

    对-2147483648，不论是直接取负，还是取绝对值，都会得到2147483648，但因为溢出了，所以最后还是回到了-2147483648 故用long long。

    除法包括整数部分和小数部分，整数部分直接除就可以。小数部分因为需要判断是否是循环小数，所以需要一个哈希表来保存余数，检查余数r是否在哈希表中，是的话则开始循环了，将括号插入到对应的位置。

    class Solution {
public:
string fractionToDecimal(int numerator, int denominator) {
    if(numerator==0) return "0";//分子为0，直接返回
    string result;
    if(numerator<0 ^ denominator<0 ) result+='-';//判断正负
    long long n=numerator,d=denominator;
    n=abs(n);d=abs(d);//转化为正数              
    result+=to_string(n/d);  //整数部分
    long long r=n%d;    //余数r
    if(r==0) return result;
    else result+='.';
    //下面处理小数部分，用哈希表（键表示余数，值表示对应的商的位置）
    unordered_map<int,int> map;
    while(r){
        if(map.count(r)){
            result.insert(map[r],1,'(');//该余数对应的商的位置
            result+=')';//当前result末尾
            break;
        }
        map[r]=result.size();    //记录该余数对应的商的位置
        //正常运算
        r*=10;
        result+=to_string(r/d);//更新result
        r=r%d;
    }
    return result;
}
};

    LeetCode 29. Divide Two Integers

    Divide two integers without using multiplication, division and mod operator.

    If it is overflow, return MAX_INT.

    解析：

    类似快速幂的思想进行快速除法，对除数进行位操作，

    注意int（-2147483648~2147483647） 8Bytes abs(-2147483648) 结果还是 -2147483648，因为此函数要返回int值，即使用long long来运算，返回的时候仍会隐式转换，所以只能进行特判(dividend == INT_MIN && divisor == -1)。

    class Solution {

public:
int divide(int dividend, int divisor) {
    if (!divisor || (dividend == INT_MIN && divisor == -1))//边界值只能特判
        return INT_MAX;
    int sign = ((dividend < 0) ^ (divisor < 0)) ? -1 : 1;//异或可以表示仅有一个条件为真的情况
//当 dividend = INT_MIN时，-dividend会溢出，所以用 long long
    long long dvd = labs(dividend);
    long long dvs = labs(divisor);
    int res = 0;
    while (dvd >= dvs) { //除法可以转换为不断减除数，可以通过对除数和结果的位操作实现加速
        long long temp = dvs, multiple = 1;
        while (dvd >= (temp << 1)) {//每次给除数翻倍，结果也翻倍
            temp <<= 1;
            multiple <<= 1;
        }
        dvd -= temp;//当前除数的倍数无法乘2的时候，可以做减法更新结果，然后等待循环下一次除数从1开始再次翻倍
        res += multiple;
    }
    return sign == 1 ? res : -res; 
}
};
