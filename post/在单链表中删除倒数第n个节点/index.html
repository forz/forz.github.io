在单链表中删除倒数第N个节点

June 25, 2017

LeetCode 19. Remove Nth Node From End of List#

Given a linked list, remove the nth node from the end of list and return its head.

For example,

Given linked list: 1->2->3->4->5, and n = 2.

After removing the second node from the end, the linked list becomes 1->2->3->5.

Note:

Given n will always be valid. Try to do this in one pass.

解析：

根据题意，不会出现链表总数不够n的情况。

first指针先移动n步，若此时first指针为空，则表示要删除的是头节点，此时直接返回head->next即可。

如果first指针不为空，则将两个指针一起移动，直到first指针指向最后一个节点，令second->next=second->next->next即可删除第n个节点。

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode *removeNthFromEnd(ListNode *head, int n) {  
        if(head == NULL || head->next == NULL)  //健壮性
            return NULL;  
        ListNode * first = head;   //快指针
        ListNode * second = head;  //慢指针
        for(int i = 0;i < n;i++)  
            first = first->next;  
        if(first == NULL)  //如果链表正好n个节点，那么直接返回
            return head->next;  

        while(first->next != NULL)  //超过n个节点，while循环
        {  
            first = first->next;  
            second = second->next;  
        }  

        second->next = second->next->next;  //删除second->next
        return head;  
    } 
};
